A website for people to record buyins for their poker homegames. As players cash
out, record their deltas. Once everyone has their delta recorded, we then
generate a payout structure that tells you who in the game that you should
request from.

To start, this service will be very simple. Simple google login and auth flow,
each user gets their own dashboard where they can create new games and view
existing ones. Should be a simple way to create the ledger but this needs to be
able to be continuously updated as the game is active.
